Unstage added files: git restore --staged [FILE_NAME.extension]

Restore to a previous commit: git restore [commit]

Stash uncommited changes: git stash

untsash uncommited changes: git stash pop

Link your local project to a git repository: git remote add origin [REPOSITORY_URL]

If a branch does not exists in git reposiory and you want to create branch and push it: git push --set-upstream origin [BRANCH_NAME]

Create New Branch in local: git branch [BRANCH_NAME]
