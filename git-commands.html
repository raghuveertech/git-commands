Unstage added files: git restore --staged [FILE_NAME.extension]

Restore to a previous commit: git restore [commit]

Stash uncommited changes: git stash

untsash uncommited changes: git stash pop

Link your local project to a git repository: git remote add origin [REPOSITORY_URL]

Create New Branch in local: git branch [BRANCH_NAME]

If a branch does not exists in git reposiory and you want to create branch and push it: git push --set-upstream origin [BRANCH_NAME]

How to revert code in repository without losing it?
1. go to previous commit: git reset [PREV_COMMIT]
2. stash the commit that is causing issue: git stash
3. now, for push the prev commit to reposiory: git push -f
4. now, get back our changes in local: git stash pop

Fetch all branches: git fetch

Fetch all branches incluing deleted branches: git fetch --all --prune

merge all un pushed commits into a single commit -
1. git reset --soft HEAD~2 (number of commits to be merged)
2. git commit -m "new commit message"
3. git push -f
